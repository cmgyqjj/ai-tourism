# 同行人关系选择页面 (trip-questions-1)

## 功能描述
这是行程创建流程的第一个问题页面，用户需要选择与同行人的关系类型。

## 页面特性
- 多选功能：用户可以选择多个关系类型
- 状态保持：选择状态会保存到本地缓存
- 进度指示：显示当前步骤 (1/13)
- 响应式设计：支持不同屏幕尺寸

## 选项列表
1. 朋友/闺蜜 👥
2. 情侣/夫妻 ❤️
3. 公司同事 👔
4. 亲子家庭 👨‍👩‍👧‍👦
5. 其他

## 使用方法
1. 点击任意选项进行选择/取消选择
2. 至少选择一个选项才能进入下一步
3. 选择完成后点击"下一步"按钮

## 技术实现
- 使用微信小程序原生框架
- 本地缓存保存用户选择
- 响应式网格布局
- 状态管理优化
